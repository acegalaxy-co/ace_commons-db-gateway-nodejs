# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in `@acegalaxy/db-gateway`, please
**do not** open a public GitHub issue.

Email **security@acegalaxy.co** with:

- A description of the issue and its impact
- Steps to reproduce (PoC welcome)
- Affected versions
- Your contact info for follow-up

We will acknowledge your report within **48 hours** and provide a timeline for
fix and disclosure.

## Supported versions

Only the latest minor of the `0.x` line receives security fixes during the
pre-1.0 phase. Once `1.0` ships, the latest two minor versions will be
supported.

## Disclosure

We follow coordinated disclosure. Once a fix is released, we will publish a
GitHub Security Advisory crediting the reporter (unless they prefer to remain
anonymous).
